ميسور - صفحة هبوط بسيطة

ملف المشروع يحتوي على صفحة هبوط من صفحة واحدة (RTL) جاهزة للعرض محليًا.

الملفات:
- `index.html` — الصفحة الرئيسية بالـ HTML.
- `css/styles.css` — أنماط الصفحة.
- `js/script.js` — تفاعلات بسيطة (السنة، التمرير الناعم، إرسال النموذج الوهمي).
- `assets/logo.svg` — شعار بسيط.
- `assets/hero-illustration.svg` — صورة توضيحية.

التشغيل:
افتح `index.html` في المتصفح (انقر مزدوج أو استخدم خادم محلي).

تخصيص سريع:
- عدل تفاصيل الاتصال في قسم `#contact` في `index.html`.
- لتوصيل نموذج إلى بريدك أو API، استبدل القسم الوهمي في `js/script.js` بطلب شبكة إلى خادمك.

مشروع مميز:
- "تطبيق ميسور": تطبيق يحول دفاتر الديون والعربون والفواتير إلى سجلات رقمية منسقة عبر نقاط البيع، مع متابعة دقيقة للديون والمدفوعات، ودفع آمن عبر NFC.

ملاحظات الوصول:
- الصفحة تستخدم اتجاه RTL وكتابة عربية.
- الحقول في النموذج تحتوي على سمات HTML5 الأساسية (`required`, `type=email`).

إذا أردت، أستطيع:
- ربط النموذج بخادم (PHP/Node/Python) أو خدمة مثل Formspree.
- إضافة نسخة إنجليزية.
- تهيئة كـ PWA أو إضافة تحليلات.
